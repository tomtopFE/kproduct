Wi-Fi Door Sensor

Koogeek Wi-Fi Door Sensor is compatible with Amazon Alexa/ Google Assistant. It can remote control ON/OFF status, supports App notification and door open/close time record.

It is compatible with Alexa and Google Assistant, so you can know the status of doors/windows by giving voice commands: “Alexa, is the door open?"
